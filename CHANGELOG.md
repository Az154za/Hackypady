## [1.0.0] - 2025-12-11

### Added
- Initial release of Hackypady mini keyboard
- 12-key programmable layout
- QMK firmware support
- VIA configurator compatibility
- RGB underglow with multiple effects
- Rotary encoder support
- OLED display integration
- 4 default layers (Base, Function, Gaming, Media)
- Comprehensive documentation
  - README with full feature list
  - BUILD_GUIDE for assembly instructions
  - KEYMAP_GUIDE for customization
  - CONTRIBUTING guidelines
- Example keymaps for different use cases
- Configuration files (config.h, rules.mk)
- MIT License

### Features
- 1ms polling rate for low latency
- N-Key Rollover (NKRO) support
- Per-key RGB lighting with layer indication
- Encoder rotation context-aware per layer
- USB-C connectivity
- Cross-platform support (Windows, macOS, Linux)
- Fully customizable key assignments
- Macro recording and playback
- Multiple RGB lighting effects:
  - Static
  - Breathing
  - Rainbow
  - Reactive
  - Wave
  - Custom programmable effects

### Documentation
- Complete build guide with step-by-step instructions
- Detailed keymap customization guide
- Hardware component specifications
- Troubleshooting section
- Example configurations for various use cases:
  - Programming
  - Video editing
  - Gaming
  - Streaming
  - Music production
  - System administration
